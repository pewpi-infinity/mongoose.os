## Important Concepts

### Machine Within a Machine
The entire system is recursive:
- mongoose.os is a machine
- Contains 287 machines (repos)
- Each repo can contain sub-machines (components)
- Sub-machines can contain modules
- Everything is modular and interconnected

### Input Steering
Machines are "steered" not "programmed":
- Provide input direction
- Machine builds itself accordingly
- Like steering a car, not building it each time
- Add-ons modify capabilities

### Token Licensing
Each machine is licensed:
- Token holders have access
- 3031 tokens currently generated
- Future: Token marketplace
- Tokens grant machine control
